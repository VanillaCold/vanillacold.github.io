Sporigins


About Sporigins

Sporigins is a new 2D evolution game that I've been developing.
Using the Sporigins Cell Creator, players can create their own cells and share them with others, and these cells can be played as in-game.
Read more on the dedicated page for Sporigins here.

Improved Hologram Scout


About the Mod

This Spore mod adds the ability to beam down onto space-stage planets, allowing for close examination of planets and their ecosystems.
The player can then use some custom abilities, alongside their creature combat abilities, to interact with the planet.
The mod is not released yet, but you can see the source code here.
